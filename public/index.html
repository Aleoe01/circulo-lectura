¿Por qué leemos?

Leemos para encontrarnos, con otros y con nosotros.
Leemos para imaginar, para crear mundos, para vivir otras experiencias.
Leemos para atravesar tempestades, miedos, para acurrucarnos y envolvernos en recuerdos.
Leemos para vivir la felicidad.
Leemos para saborear palabras nuevas.
Leemos para sentirnos vivos.
